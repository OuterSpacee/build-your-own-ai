# Contributing to Build Your Own AI

Thank you for helping make this the go-to resource for building AI projects from scratch.

## How to Add a Tutorial

1. **Fork** this repository
2. **Add** your tutorial to the correct category in `README.md`
3. Follow the **existing table format**
4. **Submit** a pull request

## Tutorial Criteria

A tutorial should:

- **Teach building from scratch** - not just using an existing tool/API
- **Be step-by-step** - clear instructions, not just a code dump
- **Be accessible** - runnable by an intermediate developer
- **Be free to access** - paywalled content should be clearly marked
- **Be maintained** - code should work with current versions

## Format

```markdown
| [Tutorial Title](https://url.com) | Primary Tech Stack |
```

- Tutorial title should be descriptive and concise
- Tech stack should list the main language/framework used
- Tutorials are listed roughly by quality/comprehensiveness within each category

## New Categories

Propose a new "Build Your Own..." category if:
- It has at least 5 quality tutorials available
- It covers a distinct AI project type
- It's broadly useful to developers

## Reporting Issues

- **Dead link?** Open an issue with the category and tutorial name
- **Tutorial outdated?** Open an issue or suggest a replacement
- **Wrong category?** Submit a PR to move it
